Text input for auth and profile forms.

```jsx
<Input placeholder="Email" type="email" />
```

- **Background:** Glass White (`#FFFFFF`)
- **Text:** Navy (`#0D1B3E`)
- **Placeholder:** Slate Muted (`#6B7FA3`)
- **Focus:** `1.5px solid #4364F7` (Quest Blue)
- **Radius:** md (12px), padding 16px

**Design spec:** `DESIGN.md` (Harbour Electric)
