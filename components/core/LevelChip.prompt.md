Compact indigo level pill. Appears on profile header (overlay on avatar) and quest feed header (next to avatar).

```jsx
<LevelChip level={4} />
<LevelChip level={7} compact />
```

**Use compact:** when overlaid on Avatar (bottom-right corner of the avatar circle)
**Use default:** when shown inline in a header row next to the avatar

**Never:** use category colors for level. Level is an earned/indigo concept — Local Signal only.
