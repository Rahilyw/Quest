# Quest! UI Guidelines

See the monorepo design spec: [`../DESIGN.md`](../DESIGN.md)

This Figma Make prototype implements the **Harbour Electric** direction:

- **Quest Sky** `#E8F3FF` — backgrounds
- **Quest Blue** `#4364F7` — brand, CTAs, active tab pill
- **City Orange** `#FF6B35` — city badge, notifications
- **Poppins** — typography (web)

Keep mock data in `src/app/data/mock.ts`. Tab structure must match mobile: Explore · Quests · Rankings · Badges · Profile.
